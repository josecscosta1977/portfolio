CãoQri - Automação Comercial - Software

O projeto de Automação Comercial(software) foi desenvolvido para o Programa de Extensão Tecnológica. Tendo como cliente a empresa CãoQri, o software será usado para os clientes da loja preencherem um formulário e retirar um brinde em uma caixa conectada ao sistema e que abrirá quando as informações forem submetidas ao banco de dados.
